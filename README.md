# Social-Compass-PBL-Final
Интердисциплинарен проект
Социален Компас: Робинзон Крузо и AIТова е интерактивно уеб приложение (PWA), разработено като част от PBL проект. То цели да помогне на потребителите да балансират времето си пред екрана, използвайки метафори от романа "Робинзон Крузо" на Даниел Дефо. Функционалностi:
Таймер за дигитален престой: Измерва времето онлайн.
Хормонални индикатори: Визуализира нивата на Кортизол (стрес) и Ендорфини (активност).
Крузо Аларм: Предлага физически мисии след определен период на застояване.
Етичен филтър: Оценява риска от социална изолация чрез казуса "Адам Рейн".
Кризисен панел: Насочва към реална помощ при нужда.
Технологии: HTML5 / CSS3 (Tailwind CSS)Vanilla JavaScriptService Workers (PWA)
